# ⚽ 足球数据分析系统

一个专业的足球比赛数据分析与预测系统，支持赔率、凯利指数、支持率等多维度数据分析。

## ✨ 功能特性

- 📊 **数据概览**：实时统计已添加比赛、已完成比赛、预测正确率
- 🏆 **比赛管理**：添加、编辑、删除比赛信息
- 📉 **凯利指数**：查看和分析凯利指数变化趋势
- 📈 **赔率分析**：追踪初赔和即赔的变化
- 👥 **支持率统计**：可视化主队、客队支持率
- 🎯 **智能预测**：基于多维度数据的综合预测分析
- 📱 **移动端优化**：完美适配手机浏览器，支持手势操作
- 💾 **数据持久化**：使用localStorage本地存储，数据不丢失
- 📤 **数据导入导出**：支持JSON格式的数据备份和恢复

## 📖 使用说明

1. 点击"比赛管理" → "添加比赛"
2. 填写比赛基本信息（联赛、时间、对阵双方）
3. 可选填写初赔、初凯利、支持率数据
4. 选中比赛后，查看四个标签页的数据分析
5. 点击"修改"按钮更新实时数据
6. 查看"预测"标签页获取综合分析结果

## 📊 数据说明

### 赔率数据
- 初赔：比赛开始前的初始赔率
- 即赔：当前的即时赔率

### 凯利指数
- 范围：0-2之间
- 理论值：接近1表示赔率合理

### 支持率
- 主胜、平局、客胜的百分比
- 总和应为100%

## 🔒 数据安全

- 所有数据存储在浏览器本地
- 不会上传到服务器
- 建议定期使用"数据导出"功能备份
